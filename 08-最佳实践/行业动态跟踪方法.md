# 行业动态跟踪方法

## 日常信息获取（每天 15 分钟）

### 早间浏览
- 36氪/亿欧网首页 — 快速扫描行业新闻
- 知乎低空经济话题 — 热门讨论
- 微信公众号推送 — 订阅号精选

### 重点关注
- 政策发布和解读
- 企业融资和产品发布
- 技术突破和学术进展
- 商业化运营案例

## 深度研究（每周 2-3 小时）

### 研报精读
- 证券研报：慧博投研搜索"低空经济"
- 行业报告：赛迪、前瞻、艾瑞最新报告
- 国际报告：McKinsey、Roland Berger

### 论文追踪
- arXiv 每周新论文（cs.RO、cs.AI）
- Papers With Code 热门项目
- 顶会论文（ICRA、CVPR、NeurIPS）

### 企业动态
- 大疆、亿航、华为等官网
- 企业财报和投资者关系页面
- 产品发布会和技术博客

## 信息整理

### 知识卡片化
每条有价值的信息都提炼为知识卡片：
- 主题标签
- 核心要点（1-2 句话）
- 来源链接
- 日期

### 分类归档
```
行业动态/
├── 政策法规
├── 企业动态
├── 技术突破
├── 融资事件
└── 商业案例
```

## 推荐工具

### 信息聚合
- **RSS 阅读器**：Feedly、Inoreader
- **稍后阅读**：Pocket、Instapaper
- **笔记工具**：Notion、Obsidian、语雀

### 知识管理
- **卡片笔记**：Obsidian、Logseq
- **思维导图**：XMind、幕布
- **项目管理**：Notion、Trello

## 信息源更新

### 定期检查（每月一次）
- 订阅的公众号是否还在更新
- 关注的企业是否有新动态
- 学术会议是否有新论文
- 开源项目是否有更新

### 扩展信息源
- 参加行业会议和展览
- 加入行业社群和论坛
- 关注新的研究机构和企业
- 与同行交流信息

## 信息到内容的转化

```
信息获取 → 知识提炼 → 内容生成 → 短视频发布
```

每条有价值的信息都应该走完这个流程：
1. 提炼为知识卡片（存入 `09-知识提炼/知识卡片/`）
2. 选择适合的卡片生成短视频脚本（存入 `10-短视频素材/脚本库/`）
3. 定期更新选题库（`10-短视频素材/选题库.md`）
