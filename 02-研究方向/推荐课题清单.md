# 推荐课题清单

适合个人研究者或小团队切入的低空经济 AI 课题。

## 入门级课题（1-3 个月）

### 课题 1：无人机航拍图像目标检测
- **内容**：基于 YOLO 系列模型，训练无人机视角的目标检测模型
- **数据集**：VisDrone Dataset
- **技术栈**：Python + PyTorch + Ultralytics YOLO
- **产出**：可部署的检测模型 + 技术报告

### 课题 2：无人机路径规划算法对比
- **内容**：实现并对比 A*、RRT、RRT*、D* 等经典路径规划算法
- **工具**：PythonRobotics / PathPlanning 开源项目
- **产出**：算法可视化 demo + 性能对比报告

### 课题 3：低空气象对飞行安全的影响分析
- **内容**：收集气象数据，分析风速、降水等对无人机飞行的影响
- **数据源**：气象局公开数据
- **产出**：数据分析报告 + 风险评估模型

## 进阶级课题（3-6 个月）

### 课题 4：基于强化学习的无人机避障
- **内容**：在仿真环境中训练无人机自主避障策略
- **工具**：AirSim/Gazebo + Stable-Baselines3
- **技术栈**：Python + PyTorch + RL 框架
- **产出**：仿真 demo + 训练日志分析

### 课题 5：低空空域冲突检测算法
- **内容**：设计基于 ML 的飞行冲突检测与预警算法
- **数据**：模拟多机飞行轨迹数据
- **产出**：冲突检测系统原型

### 课题 6：无人机视觉 SLAM 系统搭建
- **内容**：基于 ORB-SLAM3 或 VINS-Fusion 搭建视觉 SLAM 系统
- **硬件**：无人机 + 单目/双目相机
- **产出**：可运行的 SLAM 系统 + 定位精度评估

### 课题 7：多传感器融合感知系统
- **内容**：融合摄像头、激光雷达、IMU 数据进行环境感知
- **工具**：ROS + PCL + OpenCV
- **产出**：融合感知 demo

## 高级课题（6-12 个月）

### 课题 8：城市低空数字孪生平台原型
- **内容**：构建城市级低空数字孪生仿真环境
- **工具**：Unity/Unreal + AirSim + GIS 数据
- **产出**：可视化仿真平台

### 课题 9：多无人机协同任务分配
- **内容**：基于 MARL 或优化算法实现多机任务分配
- **技术**：多智能体强化学习 / 混合整数规划
- **产出**：协同调度系统原型

### 课题 10：低空大模型应用探索
- **内容**：探索 LLM/VLM 在低空场景的应用（任务规划、异常分析等）
- **技术**：GPT-4V API / 开源多模态模型
- **产出**：应用 demo + 可行性分析报告

## 选题建议

1. **从兴趣出发**：选择最吸引你的方向，保持学习动力
2. **从小做起**：先完成入门课题，建立信心和基础
3. **注重实践**：每个课题都要有可运行的代码或 demo
4. **记录过程**：每个课题的学习过程都可以提炼为知识卡片和短视频素材
