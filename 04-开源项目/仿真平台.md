# 仿真平台

## AirSim（微软）

### 概述
基于 Unreal Engine 的高保真无人机/车辆仿真平台，专为 AI 训练设计。

### GitHub
https://github.com/microsoft/AirSim

### 核心功能
- 高保真视觉渲染（接近真实照片）
- Python/C++ API
- 多机仿真支持
- 深度图、分割图、法线图等多模态数据
- 与 ROS 集成

### 安装
```bash
pip install airsim
```

### 使用示例
```python
import airsim

client = airsim.MultirotorClient()
client.confirmConnection()
client.enableApiControl(True)
client.armDisarm(True)

# 起飞
client.takeoffAsync().join()

# 移动
client.moveToPositionAsync(10, 10, -5, 5).join()

# 获取图像
responses = client.simGetImages([
    airsim.ImageRequest("0", airsim.ImageType.Scene)
])
```

### 典型应用
- 训练视觉避障模型
- 测试路径规划算法
- 生成训练数据集
- 验证自主飞行策略

## Gazebo

### 概述
机器人仿真平台，与 ROS 深度集成，是机器人研究的标准仿真环境。

### 官网
https://gazebosim.org

### 核心特点
- 物理引擎仿真（ODE、Bullet、Simbody）
- 传感器仿真（相机、激光雷达、IMU、GPS）
- ROS 原生支持
- 丰富的模型库（Gazebo Fuel）

### 与 AirSim 对比
| 维度 | AirSim | Gazebo |
|------|--------|--------|
| 视觉效果 | 高保真 | 一般 |
| 物理仿真 | 基础 | 精确 |
| ROS 集成 | 需要桥接 | 原生支持 |
| AI 训练 | 优秀 | 一般 |
| 学习曲线 | 较平缓 | 较陡 |
