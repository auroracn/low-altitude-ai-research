# 路径规划算法

## PythonRobotics

### 概述
机器人路径规划算法集合，包含大量可运行的 Python 示例。

### GitHub
https://github.com/AtsushiSakai/PythonRobotics

### 包含算法
| 类别 | 算法 |
|------|------|
| 路径规划 | A*、D*、RRT、RRT*、PRM、Informed RRT* |
| 局部规划 | DWA、LQR、MPC |
| SLAM | EKF-SLAM、FastSLAM |
| 路径跟踪 | Pure Pursuit、Stanley |
| 状态估计 | EKF、UKF、粒子滤波 |

### 特点
- 代码简洁，注释清晰
- 每个算法独立，可单独运行
- 可视化效果好
- 非常适合学习

## PathPlanning

### 概述
经典路径规划算法的 Python 实现集合。

### GitHub
https://github.com/zhm-real/PathPlanning

### 包含算法
- A*、Dijkstra
- RRT、RRT*、Informed RRT*
- D*、D* Lite
- Potential Field
- PRM

## 学习路径

### 入门
1. 学习 A* 算法原理
2. 运行 PythonRobotics 中的 A* 示例
3. 修改参数观察效果变化

### 进阶
1. 学习 RRT/RRT* 算法
2. 理解采样-based 规划的优缺点
3. 尝试在 3D 空间中实现

### 实践
1. 在 AirSim/Gazebo 中集成路径规划算法
2. 添加动态障碍物，测试重规划能力
3. 对比不同算法的性能（时间、路径长度、成功率）

## 算法选择指南

| 场景 | 推荐算法 |
|------|---------|
| 静态已知环境 | A*、Dijkstra |
| 高维空间 | RRT、RRT* |
| 动态环境 | D* Lite、MPC |
| 实时避障 | DWA、VFH |
| 多机协同 | 分布式规划算法 |
