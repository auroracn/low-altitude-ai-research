# 飞控系统

## PX4 Autopilot

### 概述
PX4 是最流行的开源无人机飞控系统，由 Dronecode 基金会支持，广泛用于学术研究和商业产品。

### GitHub
https://github.com/PX4/PX4-Autopilot

### 核心特点
- 支持多旋翼、固定翼、VTOL、Rover
- 模块化架构，易于扩展
- 丰富的传感器支持
- MAVLink 通信协议
- Gazebo/ AirSim 仿真支持

### 关键模块
| 模块 | 功能 |
|------|------|
| EKF2 | 扩展卡尔曼滤波，姿态估计 |
| Navigator | 航点导航、任务执行 |
| Flight Tasks | 飞行任务管理 |
| Commander | 飞行状态机 |
| Mavlink | 地面站通信 |

### 快速开始
```bash
# 仿真启动
make px4_sitl_default gz_x500

# 连接 QGroundControl 地面站
```

## ArduPilot

### 概述
另一个主流开源飞控，社区非常活跃，支持机型最全面。

### GitHub
https://github.com/ArduPilot/ardupilot

### 核心特点
- 支持 Copter、Plane、Rover、Sub、Tracker
- Mission Planner 功能强大的地面站
- Lua 脚本扩展
- 丰富的教程和文档

### 与 PX4 对比
| 维度 | PX4 | ArduPilot |
|------|-----|-----------|
| 架构 | 模块化，uORB 消息总线 | 传统架构 |
| 仿真 | Gazebo、AirSim | SITL、Gazebo |
| 地面站 | QGroundControl | Mission Planner |
| 社区 | 学术导向 | 爱好者/商业导向 |
| 学习曲线 | 较陡 | 较平缓 |

## Betaflight

### 概述
针对 FPV 竞速无人机的开源固件，注重飞行性能。

### GitHub
https://github.com/betaflight/betaflight

### 适用场景
- FPV 竞速无人机
- 穿越机
- 高机动性飞行

## 学习建议

1. **入门推荐 ArduPilot**：文档丰富，社区活跃，学习资源多
2. **研究推荐 PX4**：代码架构更现代，适合二次开发
3. **FPV 推荐 Betaflight**：专注竞速场景
