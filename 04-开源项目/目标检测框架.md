# 目标检测框架

## YOLO 系列（Ultralytics）

### 概述
最流行的实时目标检测框架，YOLOv8/v11 是最新版本，广泛用于无人机场景。

### GitHub
https://github.com/ultralytics/ultralytics

### 安装
```bash
pip install ultralytics
```

### 使用示例
```python
from ultralytics import YOLO

# 加载预训练模型
model = YOLO("yolov8n.pt")

# 推理
results = model("drone_image.jpg")

# 训练自定义数据集
model.train(data="visdrone.yaml", epochs=100)
```

### 版本演进
| 版本 | 特点 |
|------|------|
| YOLOv5 | 成熟稳定，部署方便 |
| YOLOv8 | 架构改进，精度提升 |
| YOLOv11 | 最新版本，性能最优 |

## YOLOv6（美团）

### 概述
美团开源的工业级目标检测框架，注重部署效率。

### GitHub
https://github.com/meituan/YOLOv6

### 特点
- 工业级优化
- TensorRT 部署友好
- 量化支持

## PaddleDetection（百度）

### 概述
百度飞桨目标检测套件，支持多种检测算法。

### GitHub
https://github.com/PaddlePaddle/PaddleDetection

### 特点
- 中文文档完善
- 支持国产硬件（昇腾、寒武纪）
- 丰富的预训练模型

## MMDetection（OpenMMLab）

### 概述
OpenMMLab 目标检测工具箱，模块化设计，适合学术研究。

### GitHub
https://github.com/open-mmlab/mmdetection

### 特点
- 模块化，易于组合
- 支持多种检测算法
- 学术论文配套代码多

## 无人机场景推荐

| 框架 | 推荐场景 |
|------|---------|
| Ultralytics YOLOv8 | 实时检测、快速原型 |
| YOLOv6 | 边缘部署、工业应用 |
| PaddleDetection | 国内项目、国产硬件 |
| MMDetection | 学术研究、算法对比 |

## 数据集

### VisDrone
- 无人机视角目标检测大规模数据集
- 包含行人、车辆、自行车等 10 类目标
- GitHub: https://github.com/VisDrone/VisDrone-Dataset

### 自定义数据集
1. 使用 LabelImg / CVAT 标注工具
2. 转换为 YOLO 格式
3. 划分训练集/验证集/测试集
