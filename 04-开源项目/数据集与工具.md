# 数据集与工具

## 无人机视觉数据集

### VisDrone
- **内容**：无人机视角目标检测/跟踪/人群计数大规模数据集
- **类别**：行人、自行车、汽车、货车、三轮车、公交车、摩托车、板车等
- **规模**：约 10,209 帧图像
- **GitHub**：https://github.com/VisDrone/VisDrone-Dataset
- **适用**：目标检测、多目标跟踪、人群计数

### UAV123
- **内容**：无人机视角目标跟踪数据集
- **规模**：123 个视频序列
- **适用**：单目标跟踪研究

### CARLA
- **内容**：自动驾驶仿真环境，可用于低空场景扩展
- **官网**：https://carla.org
- **适用**：感知算法训练

## 开发工具

### DroneKit
- **用途**：无人机应用开发 Python/Android API
- **官网**：https://dronekit.io
- **特点**：简化 MAVLink 通信，快速开发无人机应用

### MAVLink
- **用途**：无人机通信协议
- **特点**：轻量级，广泛支持
- **文档**：https://mavlink.io

### DJI SDK
| SDK | 用途 |
|-----|------|
| DJI Mobile SDK | 移动端无人机应用开发 |
| DJI Windows SDK | Windows 端开发 |
| DJI Payload SDK | 负载开发 |
| DJI Cloud API | 云端管理 |

### Tello SDK
- **用途**：Tello 无人机编程入门
- **Python 库**：djitellopy
- **特点**：简单易用，适合学习

```python
from djitellopy import Tello

tello = Tello()
tello.connect()
print(tello.get_battery())

tello.takeoff()
tello.move_forward(50)
tello.rotate_counter_clockwise(90)
tello.land()
```
