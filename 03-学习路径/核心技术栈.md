# 核心技术栈

## 1. 无人机自动驾驶

### 路径规划
| 算法 | 类型 | 特点 |
|------|------|------|
| A* | 搜索算法 | 经典全局规划，适合静态环境 |
| RRT | 采样算法 | 快速探索，适合高维空间 |
| RRT* | 采样算法 | RRT 改进，渐近最优 |
| D* | 增量算法 | 动态环境路径重规划 |
| PRM | 采样算法 | 概率路线图，适合多查询 |

### 避障算法
- **VFH（Vector Field Histogram）**：基于直方图的实时避障
- **DWA（Dynamic Window Approach）**：动态窗口避障
- **深度学习避障**：端到端学习避障策略

### SLAM
| 方法 | 代表项目 | 特点 |
|------|---------|------|
| 视觉 SLAM | ORB-SLAM3 | 单目/双目/RGB-D |
| 激光 SLAM | LOAM / LIO-SAM | 高精度，计算量大 |
| 视觉惯性 SLAM | VINS-Fusion | 视觉 + IMU 融合 |

## 2. 计算机视觉

### 目标检测
| 框架 | 特点 | 适用场景 |
|------|------|---------|
| YOLOv8/v11 | 实时检测，精度高 | 无人机实时检测 |
| YOLOv6 | 美团开源，工业级 | 物流、巡检 |
| PaddleDetection | 百度飞桨，中文社区 | 国内项目 |
| MMDetection | OpenMMLab，模块化 | 学术研究 |

### 图像分割
- **语义分割**：FCN、U-Net、DeepLab
- **实例分割**：Mask R-CNN、YOLACT
- **全景分割**：Panoptic-DeepLab

### 多传感器融合
- 视觉 + 激光雷达融合
- 视觉 + IMU 融合
- 多模态特征融合网络

## 3. 空域管理 AI

### 核心算法
- **冲突检测**：基于轨迹预测的冲突检测
- **流量预测**：时间序列模型（LSTM、Transformer）
- **调度优化**：组合优化、遗传算法、模拟退火

### 技术栈
- Python + PyTorch/TensorFlow
- 时空数据分析（GeoPandas、Shapely）
- 图神经网络（GNN）用于空域拓扑分析

## 4. 强化学习

### 核心算法
| 算法 | 类型 | 适用场景 |
|------|------|---------|
| DQN | 值函数 | 离散动作空间 |
| PPO | 策略梯度 | 通用，稳定 |
| SAC | Actor-Critic | 连续动作空间 |
| TD3 | Actor-Critic | 连续动作，双 Q 网络 |
| MAPPO | 多智能体 | 多机协同 |

### 框架
- **Stable-Baselines3**：RL 算法库
- **Ray/RLlib**：分布式 RL 训练
- **OpenAI Gym**：环境接口标准

## 工具链总结

```
Python + PyTorch + OpenCV + ROS + PX4/ArduPilot + AirSim/Gazebo
```

这是低空 AI 开发的核心工具链，覆盖从算法开发到仿真验证的完整流程。
