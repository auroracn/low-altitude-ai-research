# 工程实践指南

## 1. PX4 飞控系统

### 简介
PX4 是最流行的开源无人机飞控系统，支持多旋翼、固定翼、VTOL 等机型。

### 安装与使用
```bash
# 克隆仓库
git clone https://github.com/PX4/PX4-Autopilot.git
cd PX4-Autopilot

# 安装依赖
bash Tools/setup/ubuntu.sh

# 编译并启动仿真
make px4_sitl_default gz_x500
```

### 核心模块
- **飞行控制**：姿态控制、位置控制
- **导航**：航点导航、返航
- **传感器驱动**：IMU、GPS、气压计
- **MAVLink 通信**：与地面站通信协议

### 学习资源
- PX4 官方文档：docs.px4.io
- PX4 开发者指南
- B站搜索"PX4 入门教程"

## 2. ArduPilot

### 简介
另一个主流开源飞控，支持无人机、无人车、无人船，社区非常活跃。

### 安装
```bash
git clone https://github.com/ArduPilot/ardupilot.git
cd ardupilot
git submodule update --init --recursive
```

### 特点
- 支持机型更多（Copter、Plane、Rover、Sub）
- 社区活跃，教程丰富
- Mission Planner 地面站功能强大

## 3. AirSim 仿真平台

### 简介
微软开源的无人机/车辆仿真平台，基于 Unreal Engine，支持 AI 训练。

### 安装
```bash
pip install airsim
```

### 核心功能
- 高保真视觉渲染
- Python/C++ API
- 支持多机仿真
- 与 ROS 集成

### 典型应用
- 训练视觉避障模型
- 测试路径规划算法
- 验证自主飞行策略

## 4. Gazebo 仿真

### 简介
机器人仿真平台，与 ROS 深度集成，适合无人机系统开发。

### 核心特点
- 物理引擎仿真
- 传感器仿真（相机、激光雷达、IMU）
- ROS 原生支持
- 丰富的模型库

## 5. ROS 机器人系统

### 简介
Robot Operating System，机器人开发的标准框架。

### 核心概念
- **节点（Node）**：独立运行的进程
- **话题（Topic）**：发布/订阅通信
- **服务（Service）**：请求/响应通信
- **消息（Message）**：数据传输格式

### 安装
```bash
# ROS2 Humble (推荐)
sudo apt install ros-humble-desktop
```

### 学习资源
- ROS 官方教程：docs.ros.org
- 古月居 ROS 教程（B站）
- 《ROS机器人开发实践》

## 6. 大疆 SDK 开发

### Tello SDK
- 适合入门学习
- Python 控制接口
- 视频流获取

```python
from djitellopy import Tello
tello = Tello()
tello.connect()
tello.takeoff()
tello.move_forward(50)
tello.land()
```

### RoboMaster SDK
- 适合 AI 应用开发
- 支持机器视觉
- 竞赛平台

## 实践建议

1. **从仿真开始**：先在 AirSim/Gazebo 中验证，再上真机
2. **循序渐进**：先会用，再理解原理，最后做二次开发
3. **记录实验**：每次实验都记录参数和结果
4. **善用社区**：PX4/ArduPilot 社区有大量问题解答
