# 基础知识入门

## 低空经济核心概念

### 基本术语
| 术语 | 全称 | 含义 |
|------|------|------|
| UAV | Unmanned Aerial Vehicle | 无人驾驶飞行器，即无人机 |
| eVTOL | electric Vertical Take-Off and Landing | 电动垂直起降飞行器 |
| UAM | Urban Air Mobility | 城市空中交通 |
| UTM | Unmanned Traffic Management | 无人机交通管理系统 |
| AAM | Advanced Air Mobility | 先进空中交通（美国术语） |
| SLAM | Simultaneous Localization and Mapping | 同步定位与建图 |
| BVLOS | Beyond Visual Line of Sight | 超视距飞行 |

### 无人机分类
| 类型 | 重量 | 应用 |
|------|------|------|
| 微型 | <250g | 航拍、娱乐 |
| 轻型 | 250g-25kg | 巡检、测绘、物流 |
| 中型 | 25-150kg | 工业应用 |
| 大型 | >150kg | 货运、载人 |

## 开发环境搭建

### Python 环境
```bash
# 安装 Anaconda
# 创建虚拟环境
conda create -n uav-ai python=3.10
conda activate uav-ai

# 安装核心库
pip install numpy pandas matplotlib
pip install torch torchvision  # PyTorch
pip install ultralytics        # YOLO
pip install opencv-python      # OpenCV
```

### 推荐 IDE
- **VS Code**：轻量级，插件丰富
- **PyCharm**：功能全面，适合大型项目
- **Jupyter Notebook**：适合学习和实验

## 学习方法建议

1. **边学边练**：每个概念都用代码实现一遍
2. **做笔记**：将学习内容提炼为知识卡片
3. **设定小目标**：每周完成一个小项目或实验
4. **加入社区**：关注 GitHub、B站、知乎相关话题
