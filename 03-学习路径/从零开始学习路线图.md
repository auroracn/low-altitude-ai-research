# 从零开始学习路线图

## 总体规划

```
阶段一（1-2月）    阶段二（2-4月）    阶段三（4-6月）    阶段四（6月+）
基础概念入门  →   核心技术学习  →   工程实践动手  →   进阶方向探索
```

## 阶段一：基础概念入门（1-2 个月）

### 目标
- 理解低空经济全貌
- 掌握 AI/ML 基础概念
- 搭建 Python 开发环境

### 学习内容
| 主题 | 资源 | 时间 |
|------|------|------|
| 低空经济概念 | 本知识库 `01-行业概览/` | 1 周 |
| 无人机基础 | B站搜索"无人机入门" | 1 周 |
| Python 基础 | 菜鸟教程 / 廖雪峰 Python 教程 | 2 周 |
| AI/ML 基础 | 吴恩达机器学习课程（Coursera/B站） | 2-3 周 |
| 深度学习基础 | 吴恩达深度学习课程 / 李沐《动手学深度学习》 | 2-3 周 |

### 里程碑
- [ ] 能用自己的话解释低空经济是什么
- [ ] 能用 Python 完成基本数据处理
- [ ] 理解神经网络的基本原理

## 阶段二：核心技术学习（2-4 个月）

### 目标
- 掌握计算机视觉基础
- 理解路径规划算法
- 了解空域管理系统

### 学习内容
| 主题 | 资源 | 时间 |
|------|------|------|
| 计算机视觉 | CS231n（斯坦福）/ OpenCV 官方教程 | 3-4 周 |
| 目标检测 | YOLO 系列论文 + Ultralytics 实践 | 2-3 周 |
| 路径规划算法 | PythonRobotics 项目学习 | 2-3 周 |
| 强化学习基础 | David Silver 强化学习课程 / Spinning Up | 3-4 周 |
| UTM 系统概念 | 本知识库 `06-UTM空域管理/` | 1-2 周 |

### 里程碑
- [ ] 能用 YOLO 训练自定义目标检测模型
- [ ] 能实现并对比 2-3 种路径规划算法
- [ ] 理解 UTM 的核心功能和架构

## 阶段三：工程实践动手（4-6 个月）

### 目标
- 搭建无人机仿真环境
- 完成端到端的 AI 项目
- 积累工程经验

### 学习内容
| 主题 | 资源 | 时间 |
|------|------|------|
| ROS 基础 | ROS Wiki + 古月居 ROS 教程 | 3-4 周 |
| PX4/ArduPilot | 官方文档 + 开发者社区 | 3-4 周 |
| AirSim/Gazebo 仿真 | 官方示例 + 自定义场景 | 2-3 周 |
| 综合项目 | 选择入门/进阶课题完成 | 4-8 周 |

### 里程碑
- [ ] 在仿真环境中完成无人机自主飞行
- [ ] 完成至少一个完整的 AI + 无人机项目
- [ ] 项目成果可演示、可复现

## 阶段四：进阶方向探索（6 个月+）

### 目标
- 深入一个细分方向
- 跟踪前沿研究
- 产出研究成果

### 可选方向
| 方向 | 内容 |
|------|------|
| 大模型应用 | LLM/VLM 在低空场景的应用研究 |
| 集群智能 | 多无人机协同算法研究 |
| 数字孪生 | 低空仿真平台开发 |
| eVTOL 自动驾驶 | 感知-决策-控制一体化 |
| 空域管理 AI | UTM 中的 AI 算法研究 |

### 学习建议
- 阅读顶会论文（CVPR、ICRA、NeurIPS 等）
- 参与开源项目贡献
- 参加行业会议和技术社区
- 持续提炼知识卡片，积累短视频素材
