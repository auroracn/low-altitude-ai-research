# 低空经济 AI 智能化研究

## 项目说明

本项目是一个低空经济 AI 智能化方向的知识库，服务于从零开始系统学习该领域的从业者。涵盖行业概览、研究方向、学习路径、开源项目、行业企业、UTM 空域管理、学习资源、最佳实践，并内置知识提炼和短视频内容生成工作流。

## 用户画像

- 低空经济从业者，希望深入 AI 智能化方向
- AI 应用爱好者，具备基础编程能力
- 希望将学习成果转化为短视频内容

## 知识库结构

> Obsidian 用户请从 `00-知识索引.md` 开始，使用 wikilinks 跳转浏览。

```
00-知识索引.md       — Obsidian MOC 知识地图入口
01-行业概览/        — 行业背景、产业链、国家政策
02-研究方向/        — AI技术方向、国际热点、前沿交叉、推荐课题
03-学习路径/        — 从零到进阶的分阶段学习计划
04-开源项目/        — 飞控、目标检测、路径规划、仿真、数据集
05-行业企业/        — eVTOL、无人机、基础设施、供应链企业
06-UTM空域管理/     — UTM系统、国际对标、中国低空智联网
07-学习资源/        — 书籍课程、论文会议、行业报告、信息源
08-最佳实践/        — 研究选题建议、动态跟踪方法
09-知识提炼/        — 知识卡片模板、短视频脚本模板、知识卡片
10-短视频素材/      — 脚本库、选题库、制作指南
```

## 知识提炼工作流

每次学习一个主题后，按以下流程操作：

1. **提炼知识卡片**：使用 `09-知识提炼/知识提炼模板.md` 的格式，将学习内容提炼为结构化知识卡片，存入 `09-知识提炼/知识卡片/` 目录
2. **生成短视频脚本**：基于知识卡片，使用 `09-知识提炼/短视频脚本模板.md` 生成双平台脚本（抖音竖屏 + B站横屏），存入 `10-短视频素材/脚本库/` 目录
3. **更新选题库**：将新发现的选题补充到 `10-短视频素材/选题库.md`

### 知识卡片命名规范
`YYYY-MM-DD-主题关键词.md`，例如：`2026-05-09-eVTOL适航认证.md`

### 短视频脚本命名规范
`YYYY-MM-DD-平台-标题关键词.md`，例如：`2026-05-09-抖音-eVTOL适航认证.md`

## 资料收集指引

### 来源渠道
- **政策文件**：中国政府网 (gov.cn)、中国民航局 (caac.gov.cn)
- **行业报告**：赛迪研究院、前瞻产业研究院、艾瑞咨询
- **学术论文**：arXiv (cs.AI/cs.RO)、IEEE Xplore、Papers With Code
- **开源项目**：GitHub (搜索关键词: UAV AI, drone detection, path planning, UTM)
- **视频教程**：B站 (搜索: 无人机开发、PX4、ROS、目标检测)
- **行业资讯**：36氪、亿欧网、低空经济相关公众号
- **企业动态**：大疆、亿航、华为等官网

### 更新策略
- 每周跟踪行业政策和企业动态
- 每月更新学习资源和开源项目列表
- 每次学习新主题后执行知识提炼工作流
