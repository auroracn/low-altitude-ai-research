# 无人机企业

## 大疆创新（DJI）

### 基本信息
- **总部**：深圳
- **市场地位**：全球消费级无人机市场占有率超 70%

### 产品线
| 系列 | 定位 |
|------|------|
| Mavic | 消费级航拍 |
| Air | 轻量航拍 |
| Mini | 入门级 |
| Matrice | 行业应用 |
| Agras | 农业植保 |
| FPV | 竞速飞行 |

### AI 技术应用
- **视觉感知**：多方向避障系统
- **智能跟随**：AI 目标跟踪
- **航拍智能**：一键短片、智能构图
- **行业应用**：电力巡检 AI 识别、农业处方图

### 管理系统
- **AeroScope**：无人机安全飞行管理系统
- **FlightHub**：企业级无人机管理平台

## 行业趋势

1. **消费级**：大疆一家独大，竞争格局稳定
2. **工业级**：细分赛道机会多，垂直领域深耕
3. **农业级**：极飞、大疆双强格局
4. **军用转民用**：中航等传统军工企业加速布局
