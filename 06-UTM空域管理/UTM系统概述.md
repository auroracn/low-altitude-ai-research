# UTM 系统概述

## 什么是 UTM

UTM（Unmanned Traffic Management）是专门用于管理低空无人机飞行的交通管理系统，类似于有人航空的空中交通管制（ATC），但面向自动化、大规模的无人机运行。

## 核心功能

| 功能 | 说明 |
|------|------|
| 航路规划 | 自动规划安全高效的飞行路径 |
| 冲突检测与消解 | 实时检测飞行冲突并自动处理 |
| 实时监控 | 对低空飞行器的全域态势感知 |
| 动态空域分配 | 根据需求实时调整空域使用 |
| 飞行审批 | 简化飞行计划审批流程 |
| 应急处置 | 紧急情况下的快速响应 |

## UTM vs ATC

| 维度 | ATC（有人航空） | UTM（无人机） |
|------|----------------|--------------|
| 管理对象 | 有人驾驶飞机 | 无人机/eVTOL |
| 管理方式 | 人工指挥 | 自动化为主 |
| 飞行密度 | 低 | 高（潜在数万架） |
| 飞行高度 | 中高空 | 低空（<1000m） |
| 通信方式 | 语音 | 数据链路 |

## 中国 UTM 发展路径

### 现状
- 中国民航局推进低空空域管理改革
- 建立分级分类的空域管理体系
- 探索中国特色的低空智联网（LAIN）体系

### 发展阶段
1. **试点阶段**：深圳、广州等城市先行试点
2. **推广阶段**：经验复制到更多城市
3. **全国联网**：建立全国统一的低空管理平台

## 技术架构

```
应用层    —— 飞行计划、监控、调度、应急
平台层    —— 数据处理、AI 决策、仿真
通信层    —— 5G/卫星/ADS-B/Remote ID
感知层    —— 雷达、光电、射频检测
基础设施层 —— 起降场、充电站、气象站
```
