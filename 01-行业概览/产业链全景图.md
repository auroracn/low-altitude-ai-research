# 产业链全景图

## 上游：核心零部件与基础技术

### 动力系统
- **电池**：高能量密度锂电池、固态电池、氢燃料电池
  - 代表企业：宁德时代、国轩高科、亿纬锂能
- **电机/电控**：高功率密度电机、电调系统
  - 代表企业：卧龙电驱、汇川技术
- **螺旋桨/旋翼**：碳纤维复合材料桨叶
  - 代表企业：光威复材

### 飞控与导航
- **飞控芯片**：惯性导航（INS）、GNSS/北斗模块
- **传感器**：激光雷达、摄像头、毫米波雷达、超声波
- **通信模块**：5G 模组、ADS-B 应答器

### 机体结构
- **材料**：碳纤维复合材料、铝合金、工程塑料
- **设计**：气动布局、结构强度仿真

## 中游：整机制造与系统集成

### 无人机整机
| 类型 | 应用场景 | 代表企业 |
|------|---------|---------|
| 消费级无人机 | 航拍、娱乐 | 大疆、零度 |
| 工业级无人机 | 巡检、测绘、安防 | 纵横股份、极飞科技 |
| 物流无人机 | 快递配送、应急物资 | 丰翼科技（顺丰）、美团 |

### eVTOL 整机
| 构型 | 特点 | 代表企业 |
|------|------|---------|
| 多旋翼 | 结构简单，航程短 | 亿航智能 |
| 复合翼 | 航程长，效率高 | 峰飞航空、御风未来 |
| 倾转旋翼 | 速度快，技术复杂 | 沃兰特、时的科技 |

### 核心系统集成
- 飞控系统集成
- 动力系统集成
- 航电系统集成
- 地面站系统

## 下游：运营服务与应用场景

### 应用场景

| 场景 | 说明 | 成熟度 |
|------|------|--------|
| 农业植保 | 农药喷洒、播种、施肥 | 成熟 |
| 电力巡检 | 输电线路、风电叶片巡检 | 成熟 |
| 测绘地理信息 | 航测、三维建模 | 成熟 |
| 物流配送 | 城市末端配送、应急物资 | 快速发展 |
| 低空旅游 | 景区观光飞行 | 试点中 |
| 城市空中出行 | eVTOL 载人交通 | 早期 |
| 应急救援 | 医疗物资运输、搜救 | 推广中 |
| 安防监控 | 城市安防、边境巡逻 | 成熟 |

### 运营服务
- **飞行运营**：航线规划、飞行执行、机队管理
- **维修保障**：无人机/eVTOL 维修、零部件更换
- **培训服务**：飞手培训、适航审定培训
- **数据服务**：飞行数据、地理信息数据

### 基础设施
- **起降场（Vertiport）**：城市空中交通的地面节点
- **通信网络**：低空智联网（5G-A/卫星通信）
- **空管系统（UTM）**：低空空域管理核心系统
- **能源补给**：充电桩、换电站

## 产业链价值分布

```
上游零部件 (30%) → 中游整机制造 (40%) → 下游运营服务 (30%)
```

当前阶段，中游整机制造和上游核心零部件是价值最集中的环节。随着低空经济成熟，下游运营服务的占比将持续提升。
